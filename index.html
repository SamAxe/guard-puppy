guard-puppy by SamAxe

This is based on guarddog 2.6.0 from http://www.simonzone.com/software/guarddog/. The goal of this project will be bring the code up to current package versions. This is pre beta software, i.e. don't use it (unless you're going to develop and help debug it).

The aim of this code is to bring the existing guarddog 2.6.0 functionality to current distribution package versions. Right now, plans are to eventually get a beta package for Redhat and Ubuntu/Debian

Dependencies

Qt 4.8.0 -- Is what it's developed with, other versions make work QMake -- for the makefile generation Boost 1.46 -- A couple of headers, primarily the reg ex support

Install

Download the source Run qmake in the main directory. May need to help locate the boost regex library if it didn't find it
